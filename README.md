🕷️ Documentação do Projeto - Hallownest Chronicles

📌 Contexto

Este projeto foi desenvolvido como parte de uma atividade acadêmica com o objetivo de aplicar conceitos de desenvolvimento web na criação de um site temático.

O tema escolhido foi a lore de Hollow Knight, um jogo conhecido por sua narrativa indireta e rica em detalhes, que exige interpretação por parte do jogador.

A proposta do site é reunir e organizar essas informações de forma clara, facilitando o entendimento da história para quem deseja explorar mais profundamente o universo do jogo.


🎯 Objetivo

O principal objetivo do projeto é apresentar a história, personagens e eventos de Hollow Knight de maneira estruturada e acessível.

Além disso, o projeto também busca:

Praticar conceitos de desenvolvimento web
Trabalhar a organização de conteúdo
Desenvolver uma interface visual coerente com o tema
🧩 Descrição do Projeto

O site “Hallownest Chronicles — Arthur’s Journey” funciona como um guia sobre a lore do jogo.

A estrutura foi pensada para conduzir o usuário por diferentes aspectos do universo de Hallownest, como:

Contexto geral da história
Personagens importantes
Eventos e interpretações

A navegação e o visual seguem uma proposta inspirada no próprio jogo, com uma estética mais escura e misteriosa, reforçando a ambientação e a imersão.


💡 Relação Pessoal com o Projeto

A escolha desse tema também tem uma ligação pessoal.

Durante o desenvolvimento, me identifiquei com a forma como a história de Hollow Knight é construída: aos poucos, por meio de descobertas, interpretação e conexão entre partes soltas.

Esse processo se conecta com minha própria experiência ao longo do projeto, onde foi necessário aprender, testar e evoluir gradualmente até chegar ao resultado final.

Assim como no jogo, o entendimento não vem de forma imediata, mas sim através da exploração e da persistência — algo que também fez parte do meu processo de aprendizado.


🛠️ Tecnologias Utilizadas

O projeto foi desenvolvido utilizando:

HTML
CSS
JavaScript



🚀 Considerações Finais

Este projeto representa a aplicação prática de conceitos de desenvolvimento web aliada à organização de uma narrativa complexa.

Além do aprendizado técnico, também foi uma oportunidade de transformar um interesse pessoal em um projeto acadêmico estruturado, desenvolvendo tanto habilidades técnicas quanto a capacidade de organizar e apresentar informações de forma clara.
